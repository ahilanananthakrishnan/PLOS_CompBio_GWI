README

hypertrophy_fitting.py: Fits and visualizes inflammation-driven hypertrophy.

macrophage_fitting.py: Fits and visualizes macrophage kinetics.

stress_fitting.py: Fits and visualizes excitatory and inhibitory neural active stress.

Both scripts are reference Python implementations for equations used in Abaqus UMATs.

No files are written; results are printed and plotted only.
